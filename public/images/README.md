# Фоновое изображение для Hero секции

Загрузите изображение с морепродуктами в эту папку с именем:
**seafood-hero-bg.jpg**

Или измените путь в компоненте Hero.tsx на ваше изображение.

Рекомендуемые параметры:
- Формат: JPG или WebP
- Разрешение: минимум 1920x1080px
- Размер файла: до 500KB (для быстрой загрузки)
